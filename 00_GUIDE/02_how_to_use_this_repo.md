# 📌 How to Use This Repository

This repository is structured to take you from **beginner to advanced (job-ready)** in Data Analytics.

It is NOT meant for passive learning.  
It is designed for **active practice and real-world problem solving**.

---

## 🚀 Learning Flow (Follow Strictly)

You MUST follow this order:

1. **01_EXCEL**
2. **02_SQL**
3. **03_PYTHON**
4. **04_POWER_BI**
5. **05_STATISTICS**
6. **06_PROJECTS**
7. **07_PORTFOLIO_PROJECTS**
8. **08_INTERVIEW_PREP**

Skipping steps will weaken your foundation.

---

## 📂 How Each Section Works

Each main folder contains:

- Concepts (basics → advanced)
- Practice files
- Real-world tasks
- Projects (beginner → master level)

## 🧠 How to Learn Effectively

Follow this approach:

1. Learn concept
2. Apply on dataset
3. Solve problem
4. Create output (Excel / SQL / Dashboard)
5. Write insights

👉 If you skip step 5, your work is useless.

---

## 📁 What Each Project Must Contain

Every project folder should include:

- Dataset (or reference link)
- Cleaned data
- Analysis file (Excel / SQL / Python)
- Output (charts / dashboard)
- README.md

---

## 📄 Project README Format

Each project must explain:

- Problem Statement
- Dataset Description
- Approach
- Analysis Steps
- Key Insights
- Business Recommendations
